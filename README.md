# change-to-dvorak

## MacOS

To dvorak:

```bash
git clone https://github.com/odysseasarchaios/change-to-dvorak.git
cd change-to-dvorak/
chmod +x dvorak.sh
./dvorak.sh
```

To qwerty:
```bash
git clone https://github.com/odysseasarchaios/change-to-dvorak.git
cd change-to-dvorak/
chmod +x qwerty.sh
./qwerty.sh
```
